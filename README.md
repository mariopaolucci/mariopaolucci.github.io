## Work and AI: snooze tabs for firefox.

It is a strange moment for coders. I read everywhere that AI coding should be avoided. Yet for an old, rusty coder like me, the temptation is irresistible. In just a couple of days, and with no specific knowledge of javascript, I've managed to create somthing that works for me: a modification of the [Snooze Tabs]https://github.com/ramkumar-kr/snooze-tabs extension that allows for periodical refreshment. This way, I can have some tabs just pop up once a week (recurring work tasks), or once a month (like, poetry, or ["The Dream-Quest of Unknown Kadath" by H. P. Lovecraft](https://www.hplovecraft.com/writings/texts/fiction/dq.aspx) that I can re-read forever as if it was poetry.)

Am I a bad person for using AI? Probably yes. Is there a way to stop this globally apart from a massive market crash? Probably not.
